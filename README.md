# Shreyas Sharma — Portfolio

Personal portfolio website showcasing robotics, embedded systems, and ML/CV projects.

**Live:** Deployed via GitHub Pages

## Tech Stack

- Vanilla HTML / CSS / JavaScript
- GSAP 3 + ScrollTrigger (CDN)
- Web Audio API (SFX toggle)

## Features

- Boot sequence with skip-on-click
- Particle canvas background
- Blueprint mode toggle (engineering CAD aesthetic)
- SFX toggle (synthesized click sounds)
- Custom cursor with trail
- Scroll-triggered reveal animations
- Fully responsive layout

## Projects Featured

- Autonomous EV Charging Robot (ROS2 + EVS-YOLO)
- SmartNode IoT Energy Monitoring (ESP32 + MQTT + React)
- Drone Detection & Multi-Object Tracking (YOLOv8 + ByteTrack)
- 5-DOF Robotic Arm (ESP32 + PCA9685 + Python IK)
- And 10 more builds across controls, embedded, and ML

## Local Development

Open `index.html` directly in a browser — no build step required.

## Deployment

Push to GitHub and enable GitHub Pages from **Settings → Pages → Deploy from branch → main**.
